# 🖨️ Filament Price Tracker & Maker Ecosystem

A collection of community-focused tools designed to help makers optimize their 3D printing workflow and save money.

## 🚀 Featured Tools

### 💰 [Filament Price Tracker](https://filamentpricetracker.com)
The primary utility for tracking real-time price drops across major filament brands.
- **Real-time Ingestion:** Monitors prices for PLA, PETG, ASA, and more.
- **Historical Data:** Know if a "sale" is actually a good deal.
- **Brand Support:** Optimized for Bambu Lab, Hatchbox, Sunlu, and Overture.

### 🎨 [Fab Meld](https://fabmeld.com)
A visual discovery hub for 3D printing projects, technical guides, and "maker-tech" integration.
- High-quality visual galleries.
- Technical "How-to" guides for 3D printing.
- 3D Printers
- Calibration for 3D Printing.

## 🛠️ Technical Stack (The "How it Works")
While the core scrapers are private to prevent abuse, the ecosystem is built on:
- **Backend:** Python & FastAPI
- **Security:** Hardened infrastructure with a privacy-first approach to user data.
- **Automation:** Custom agents for price discovery and news aggregation.

## 🔒 Security & Privacy
As a security professional, I build tools with a **privacy-first** mindset. 
- No intrusive tracking.
- Local-first architecture wherever possible.
- Transparent data handling.
